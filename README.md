# Spotify Stats

A web application for viewing your Spotify listening statistics.

## Development

This project uses **pnpm** as the package manager. Do not use `npm` or generate `package-lock.json` files.

### Prerequisites

- Node.js (LTS version recommended)
- pnpm

### Installation

```bash
pnpm install
```

### Available Scripts

- `pnpm run dev` - Build frontend and start development server with Wrangler
- `pnpm run dev:frontend` - Start frontend development server with Vite
- `pnpm run build:frontend` - Build the frontend for production
- `pnpm run lint` - Run ESLint
- `pnpm run format` - Format code with Prettier
- `pnpm run test` - Run tests
- `pnpm run deploy` - Build and deploy to Cloudflare Workers
